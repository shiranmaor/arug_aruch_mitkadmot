# Project Context

This project is the landing page for Shiran's advanced woven wrap workshop (`סדנת ארוג ארוך למתקדמות`).

Current status:
- The first task in this branch/session was a color-palette replacement.
- This follow-up task refines the hero-to-page color transition and improves the hamburger button visual styling.
- The design flow, layout, spacing, content, RTL Hebrew structure, and JavaScript behavior were required to remain unchanged.
- The previous palette was purple/lilac-based.
- The current palette is warm yellow / golden / olive-yellow, inspired by the user's provided reference image direction.
- No new git branch was opened during this work.
- The current working branch remains the existing branch, expected and confirmed as `main`.

Implementation scope:
- Palette changes were applied primarily through the existing `:root` CSS variables in `index.html`.
- Remaining hardcoded old palette references and cool-tinted accent states were updated where relevant so the page stays visually consistent.
- The follow-up refinement kept changes limited to CSS styling and project documentation.
- The hero gradient and the lightest palette values were softened so the top area fades more naturally into the body background.
- The hamburger button styling was improved to better match the warm golden palette while keeping size, markup behavior, and drawer logic unchanged.
- Form, drawer menu, PayBox redirect, Formspree submit, Google Apps Script integration, availability logic, and edit-mode GitHub save logic were not intentionally changed.
